# Claude分析请求模板

> 复制下面的内容发给Claude

---

## 每日分析（最常用）

```
请帮我分析今日A股行情和我的持仓：

GitHub: https://github.com/ZusyLu/learn

请阅读：
- about_me.md（我的投资风格）
- portfolio.md（当前持仓）
- market_今天日期.md（今日行情）
- watchlist.md（观察清单）

分析：
1. 今日市场主线？情绪如何？
2. 我的持仓有风险吗？需要操作吗？
3. 明日预判和操作建议
4. 观察清单中有买入机会吗？
```
